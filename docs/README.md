## 文档状态约定

| 标记 | 含义 |
|---|---|
| 已实现 | 已在当前代码中存在并可通过构建或静态审阅确认 |
| 部分实现 | 主路径存在，但规则、UI、内容或验证未闭环 |
| 建议 | 本轮评审提出的产品或工程方向，尚未落地 |
| 待定 | 需要制作人、数据或试玩结论决定 |

## 单一事实来源

- 当前数值：`src/game/constants.js` 与 `src/game/data/*.js`。
- 运行时规则：`src/game/core.js`。
- 渲染与视觉：`src/game/render.js`、`src/game/sprites.js`、`src/styles.css`、`src/assets/`。
- 在线服务：`src/online/` 与根目录 `supabase-light-online.sql`。
- 本文档描述目标与约束；若实现变更，应在同一提交更新相应文档与数值表。
